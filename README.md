# פורטל למידה: ניהול הגנת סייבר בארגון

אתר זה נבנה במיוחד כדי לסייע בלמידה ותרגול לקראת הבחינה הסופית בקורס **ניהול הגנת סייבר בארגון**. 
האתר מבוסס על השוואה ואינטגרציה של כלל חומרי הקורס: המצגות, תמלולי השיעורים, סיכומים קצרים, והמבחן לדוגמה.

## 🚀 תכונות עיקריות
1. **סיכומים מקיפים לכל 14 המפגשים**: סיכומים מובנים, קריאים ומעוצבים המבוססים על ההרצאות והתמלולים.
2. **התאמה מלאה למבחן**: בכל שיעור משולבות השאלות הרלוונטיות אליו מתוך המבחן לדוגמה (כולל משוב מיידי והסברים).
3. **מצגות מובנות**: המצגת המקורית של כל שיעור מוטמעת ישירות בתחתית העמוד שלו לצפייה נוחה.
4. **כרטיסיות שינון אינטראקטיביות (Flashcards)**: כרטיסיות תרגול דו-צדדיות (תלת-מימדיות) לשינון מושגי יסוד ונושאי מפתח עם אפשרות לסינון לפי שיעור ומעקב התקדמות.
5. **סימולטור מבחן לדוגמה (שני מצבים)**:
   - **מצב למידה (תרגול)**: בדיקה מיידית של כל תשובה עם הצגת הסבר מפורט.
   - **מצב מבחן (סימולציה)**: בחינה מלאה של 22 שאלות ללא משוב בזמן אמת, הגשה בסיום, קבלת ציון מפורט, וניתוח ההצלחה לפי נושאי לימוד.
6. **חיפוש גלובלי מהיר**: חיפוש חופשי של מושגים, שאלות או מילים בכלל חומרי השיעורים והמבחן.
7. **עיצוב מודרני מותאם אישית**: ערכת נושא כהה (Dark Mode) יוקרתית כברירת מחדל עם אפשרות למעבר לערכת נושא בהירה (Light Mode).

## 🛠️ העלאה ל-GitHub והפעלה כאתר חינמי (GitHub Pages)

הפרויקט הוגדר כמאגר Git מקומי מוכן. כדי להעלות אותו ל-GitHub שלך ולהפוך אותו לאתר אינטרנט חי ונגיש מכל מקום:

1. היכנס לחשבון ה-GitHub שלך וצור מאגר (Repository) חדש וריק (ללא README, ללא .gitignore וללא רישיון). תן לו שם, למשל: `cyber-security-study`.
2. פתח את הטרמינל במחשב בתיקיית האתר והרץ את הפקודות הבאות כדי לקשר ולהעלות את הקוד:
   ```bash
   # קישור למאגר החדש שיצרת (החלף בקישור האמיתי של ה-Repository שלך)
   git remote add origin https://github.com/username/cyber-security-study.git
   
   # הגדרת ענף ראשי בשם main
   git branch -M main
   
   # העלאת הקבצים (ייתכן ותתבקש להזין שם משתמש וסיסמה/Token)
   git push -u origin main
   ```
3. לאחר שהעלאה הסתיימה, כנס בהגדרות המאגר ב-GitHub לטאב **Settings** -> **Pages**.
4. תחת **Build and deployment** -> **Source**, בחר ב-**Deploy from a branch**.
5. תחת **Branch**, בחר ב-**main** ובתיקייה **/(root)**, ולאחר מכן לחץ על **Save**.
6. תוך 1-2 דקות, GitHub Pages יפרסם את האתר שלך בכתובת ציבורית (למשל: `https://username.github.io/cyber-security-study/`) ותוכל לגלוש אליו מכל מכשיר!
